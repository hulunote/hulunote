Hulunote – an outliner note app with a built-in MCP client

Hi HN, I'm the author of Hulunote.
Hulunote is an open-source outliner note app with a built-in MCP client.

The idea is that notes shouldn't just store information — they should also be able to call tools and run AI workflows.

Recent work focuses on: - MCP reflective reasoning - connecting to a Hulunote MCP server for batch note operations - drag-and-drop node editing - syntax highlighting

Happy to answer any questions!
