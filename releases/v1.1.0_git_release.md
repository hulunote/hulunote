# 🚀 Hulunote v1.1.0 — MCP Reasoning & Major UI Upgrade

This release focuses on **AI-powered note operations and a major UI upgrade**.
Hulunote now supports **MCP reflective reasoning**, enabling AI to think through note structures and perform batch operations across your knowledge base.

---

## ✨ Highlights

### 🧠 MCP Reflective Reasoning

Hulunote now supports **reflection-based reasoning through MCP**.

AI can:

* Analyze note structures
* Perform **multi-step reasoning**
* Execute **batch note organization tasks**

This allows AI agents to **think before acting**, improving accuracy when working with large knowledge bases.

---

### 🔌 Hulunote MCP Server Integration

You can now connect directly to the **Hulunote MCP Server**.

This enables:

* AI-driven **batch note organization**
* Automated knowledge restructuring
* Programmatic note operations through MCP tools

Your notes are no longer just documents —
they become **structured data that AI can operate on**.

---

### 🖱 Major UI Upgrade

The editor interface has been significantly improved.

New capabilities include:

* **Drag-and-drop node movement**
* More fluid outliner interactions
* Improved layout and usability

Working with large outlines is now much faster.

---

### 💡 Syntax Highlighting

Code blocks now support **syntax highlighting**, making technical notes and code snippets easier to read.

---

## 🏗 Stack

Rust (Axum, SQLx) + ClojureScript + PostgreSQL

## 🎯 Vision

Hulunote continues to push toward a new workflow:

> **Notes are not static documents — they are structured knowledge that AI can reason about and operate on.**

With **MCP reasoning + tool integration**, your knowledge base becomes an **AI-operable workspace**.
